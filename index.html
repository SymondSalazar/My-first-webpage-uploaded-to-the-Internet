What is the programming?

Computer programming is the process by which the source code of computer programs is designed, encoded, cleaned and protected. The objective of programming is to create software, which will then be executed directly by the computer's hardware, or through another program.

What is Game development?

Game Development is the art of creating games and describes the design, development and release of a game. It may involve concept generation, design, build, test and release. While you create a game, it is important to think about the game mechanics, rewards, player engagement and level design.
